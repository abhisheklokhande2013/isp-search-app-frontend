ISP Listing HackerEarth Challenge

This assignment was a part of HackerEarth's Full stack application development challenge
Link to download Problem statement
Technology stack used
Front-end: Angular 6
Styling: Semantic UI
Back-end: NodeJS (MVC architecture)
ExpressJS framework
Mongoose library
Database: MongoDB Atlas
Components
ISP CRUD Component: To perform all CRUD operations
ISP Dashboard Component: To load statistic
ISP List Component: To load list of ISP's
ISP Info Component: To show detailed ISP info
Service
ISP-Service: To perform all API operations
Models
ISP-Data: To store ISP object
Future Scope
Login and Authentication component

Billing component with different payment modes
UI improvements like show stars for rating
